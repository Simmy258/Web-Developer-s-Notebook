Steps to become a web developer

  1. Learn How to search elements
    • Use Devdocs' website to study the elements
    • Use "mdn" after the thing your are searching for
  2. Implement elements
    • Make sure that you understood the elements attributes and information given on devdocs website
    • Know how to differentiate tags (the ones that need to be closed and the simple ones)
  3. Create a webssite
    • Put to practise all the elements that you have learnt that day. Don't wait to learn multiple elements to start your website

Vocabulary
Elements is a type of HTML (Hypertext Markup Language) document component
HTML (HyperText Markup Language) is the most basic building block of the Web.
